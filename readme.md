核心理念：使用ldlib2制作可视化编辑界面，使用KubeJS实现脚本自定义npcai逻辑。成为最自由功能最丰富最现代化的npc模组

版本：1.21 - 1.21.1 NeoForge

模组前置：ldlib2

模组联动：KubeJS
